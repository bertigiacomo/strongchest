# Strongchest

A Windows desktop app to store files inside an AES-encrypted, password-protected vault.
Built from scratch during the COVID-19 lockdown in 2020, when I was 15.

## Features

- Drag & drop files and folders into the encrypted vault
- Open files directly from the vault (decrypted on the fly, no manual export needed)
- Export the entire vault as a single encrypted archive, and import it on another machine
- Auto-updater with silent install support
- Company/enterprise branding profile (custom logo, name, URL, update policy)
- Storage space limit with usage graph
- Dark and light theme
- Session logging

## Tech

Visual Basic .NET · .NET Framework 4.6.1 · WinForms · AES / SHA-256 / SHA-512

## Build

Open `Strongchest.sln` in Visual Studio 2019+, restore NuGet packages, hit F5.
